# 📚 Поиск публикаций в Crossref API

Асинхронное веб-приложение на Flask для поиска научных публикаций в базе Crossref с использованием aiohttp и asyncio.


## Обзор проекта

Веб-приложение предоставляет удобный интерфейс для поиска научных публикаций в базе данных Crossref. Поддерживает два типа поиска:
- 🔍 **По автору** (параметр `query.author`)
- 📖 **По названию** (параметр `query.title`)

Приложение использует **асинхронные запросы** для быстрого получения данных и отображает информацию о публикациях в удобном карточном формате.

### Ключевые возможности

✅ Асинхронные запросы к API Crossref  
✅ Множественный поиск (несколько авторов/названий одновременно)  
✅ Красивое отображение результатов с Bootstrap  
✅ Обработка всех типов ошибок  
✅ Адаптивный дизайн для всех устройств  
✅ REST API для интеграции с другими приложениями
